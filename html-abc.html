Os titulos podem ser colocados de h1 a h6

p para parágrafos normais

Para deixar um texto forte se usa strong semanticamente se n usa b

para deixar pequeno small

Um texto riscado se usa del

Para texto em cima oi se usa sup

Para texto em baixo 10 se usa sub


para qubrar linha br

Para deixar um texto italico se usa o i

Texto marcado se usa o mark texto marcado

hr faz um linha de um lado ate o outro da tela

Para deixar um texo sublinhado se usa a tag u texto exemplo

para abreviar uma texto se usa abbr <- passe o mouse

Para colocar um endereço: rua do exemplo 321, exemplo@e-mail.com usa-se adress

Para citar algo como texto usa-se cite essa é a citaçao se usa o q

Um texto que em um momento teria um trecho

especial se usa blockquote

Para inverter o texto usa bdo, coloca-se a propriedade dir="" rtl da direita para esqueerda
ou ltr da esquerda pra direitatexto invertido

Para criar um link use a, para ir de um documento para outro ou
para hiperlink usa-se a propriedade href="" e para abrir nessa tela a
propriedade target=""que pode-se atribuir o _blank para uma nova pagina ou
_self para abrir na pagina atual

Para colocar imagem usa-se img src="para mostrar o local" width e height
para regular largura e altura, usa-se alt=alternativo para os sites de busca saberem com mais faciliade sobre o conteúdo

É sempre bom comprimir imagens, compress jpej ou png ,easy-reseize são sites para mexer no tamanho da imagem

para colocar um icone na aba do navegar dor coloque: link rel="shortcut icon" href="Local do icone" type="image/x-icon"

Para procurar icones na internet entre no site flaticon.com

Uma imagem pode ser um link, usando o image map genertor, site para gerar imagem clicavel com varios links

Table cria uma tabela, tr é posto na primeira linha, e th seriam as colunascom os titulos, td a baixo para adicionar linhas

para criar borda atributo "border"

listar nao ordenadas ul/ lista ordenada ol/ itens da lista il


iframe faz com q um site funcione dentro do seu site, src="google.com"

no link é possivel colocar um iframe no atributo target e fzr com q o link abra na sua pagina, coloque o name src="" name"de um nome para o iframe"


Para fzr formulario use form,

atributos dentro de form coloque < input type=text > para criar um campo para colocar o texto,
placeholder (marca dagua) dps do type para o texto ficar transparente atras
label faz uma legenda, exemplo coloca (nome) do lado do campo
para senha coloque password no TYPE
Para e-mail input TYPE=email

input type=submite envia o conteudo do formulario, o value= enviar o formulario
colocando o atributo required o campo vira obrigatorio


input type=radio faz com q apareçam as opçoes possiveis para mandar, colocar name e value,
isso permite apenas uma opcão, trocando o nome é possivel selecionar mais de uma escolha

input type=checkbox faz com que cada nome seja unico, ou seja varias opçoes

input type=select faz um combobox, ou seja varias opçoes em um quadradinho

textarea monta uma caixa onde se pode escrever dentro para enviar ao registro


button cria um botao


Tag audio para colocar audios tag filho source, com o atributo src="" para coloca o caminho de onde esta o audio, type="" para os formatos (mpeg, ogg, mp3, wav), colocando o atributo controls mostrara um controle de audio na tela

com controlsLits="nodownload" (l maiusculo) impede q a pessoa baixe o arquivo de audio

Tag video para colocar video tag filho source, com o atributo src="" para colocar o caminho do video, type="" para colocar os formatos (mp4, webm, ogg), com controls mostra os controles de video

width 100% ele oculpa a largura total da tela

para colocar uma thubmail se usa se usa dps do video a tag poster="", com autoplay dps do video ele toca automaticamente

para desabilitar o download controlsList="nodownload" (l maiusculo), para tirar o mini-player disablepictureinpicture
