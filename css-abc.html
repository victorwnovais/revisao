CSS (Cascading Style Sheets) ou Folhas de Estilo em Cascata, são arquivos que estilizam arquivos,

existem 3 maneiras de se atribuir o css no seu html
a primeira é incorporando: colocando como um atributo após qualquer tag;
a segunda é incorporado: introduzindo "style" no head;
a terceira é criando um arquivo css externo: atribuindo um LINK no head para o arquivo css


color: muda a cor da propriedade

background:mexe no fundo da propriedade

font-size: mexe no tamanho da propriedade baseado no que ela é

Seletores são indentificadores únicos podendo haver apenas um na página para cada elemento,
ele seleciona algum lemento para estilizar

Classes são indentificadores multiplos, podem haver vários por página para mais de um elemento,
ele selecionar o/os elementos para estilizar/los
