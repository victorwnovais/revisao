CSS (Cascading Style Sheets) ou Folhas de Estilo em Cascata, são arquivos que estilizam arquivos,

existem 3 maneiras de se atribuir o css no seu html
a primeira é incorporando: colocando como um atributo após qualquer tag;
a segunda é incorporado: introduzindo "style" no head;
a terceira é criando um arquivo css externo: atribuindo um LINK no head para o arquivo css

Margin serve para dar um espaçamento (afastamento) em qualquer elemento entre o espaço de fora da caixa e o elemento

O padding faz a mesma coisa em que o margin com relaçao a parte de dentro do elemento


text-align alinha o texto

text-decoration decora o texto

exemplos:
none tira a decoração do texto

overline coloca uma linha sobre o texto

line-through deixa o texto riscado

undeline sublinha o texto


text-transform muda todos os textos do elemento

exemplos:
uppercase transforma todas as letras para maiusculo

lowercase transforma todas as letras para minusculo

capitalize deixa a primeira palavra de cada texto sempre em maiusculo


text-indent espaça a primeira linha do texto(faz um paragrafo)

letter-space faz espaçamento entre as letras

word-spacing faz espaçamento entre palavras

line-height faz espaçamento entre as linhas
